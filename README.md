# CalcUp

Aplicación móvil tipo Duolingo enfocada en el aprendizaje de matemáticas mediante ejercicios interactivos, niveles y progreso del usuario.

## Objetivo del proyecto

Desarrollar una aplicación educativa gamificada donde los usuarios puedan practicar matemáticas de forma progresiva, mejorar sus habilidades y llevar un seguimiento de su avance.

## Funcionalidades destacadas

- Sistema completo de autenticación (registro, login y logout de usuarios)
- Sistema de aprendizaje por niveles con progresión del usuario
- Navegación estructurada mediante Navigation Graph (NavGraph)
- Carga de contenido dinámico desde JSON y renderizado en la interfaz
- Sistema de puntuación al completar niveles
- Ranking basado en el tiempo de finalización de cada nivel (scoreboard)
- Tienda interna donde los usuarios pueden gastar puntos obtenidos
- Inventario de objetos comprados dentro de la tienda
- Sistema de feedback en tiempo real (aciertos y errores durante los niveles)
